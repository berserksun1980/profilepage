# Yu Sun's Resume & Portfolio

This folder contains the interactive resume and portfolio website for Yu Sun.

## Files

- **resume.html** - The main interactive resume webpage

## How to View

### Option 1: Open directly in browser
```
file:///Users/neteasemtl/Documents/Profile/resume.html
```

### Option 2: Use the preview server
In Claude Code, use the profile-server configuration:
```
/preview or use the launch configuration
```

Then navigate to: `http://localhost:8080/resume.html`

## Features

- 📋 Professional summary and work experience
- 🎯 Expandable job details with achievements
- 🏆 Core competencies and technical skills
- 📚 Education, languages, and affiliations
- 📈 Interactive career timeline
- 🌙 Dark mode toggle
- 📱 Fully responsive design
- ✨ Smooth animations and transitions

## Navigation

Use the sticky navigation bar to switch between:
- Overview
- Experience
- Skills
- Education
- Timeline

## Customization

All content is embedded in the HTML file. To make changes:
1. Edit the relevant section in resume.html
2. Refresh the browser to see updates

Enjoy! 🎮
